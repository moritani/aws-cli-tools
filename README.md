# aws-cli-tools

aws-mfa: プロファイル[mfa]にAWS CLI用のCredentialsをセットする

aws-assume-role-with-mfa: プロファイル[mfa]のCredentialsを利用して別アカウントのAWS CLI用Credentialsをプロファイル[RoleName]にセットする

aws-assume-role: 上記2つを一度に実行する

aws-assume-role-without-mfa: MFAを利用せずに別アカウントのAWS CLI用Credentialsをプロファイル[RoleName]にセットする

※12桁のAWSアカウントIDやUserNameは自身の値やデフォルトで設定したい値を適切に設定して使用すること
